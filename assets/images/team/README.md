# Photos du bureau

Déposer ici les portraits (carrés ou proches du carré), nommés :

```
brigitte.jpg        # Brigitte Maccioni Charnallet
herve.jpg           # Hervé Charnallet
paul.jpg            # Paul Duan
jean-benoit.jpg     # Pr. Jean-Benoît Arlet
laurence.jpg        # Dr. Laurence Goux
```

Recommandations : 400×400 px, qualité 80 %, moins de 100 Ko par image. La page `notre-histoire.html` les insère dans un cercle de 120 px — si un fichier manque, les initiales (BC, HC, PD, JA, LG) restent affichées proprement.

## Sources rapides

- LinkedIn (bouton droit → "Enregistrer l'image" sur la photo de profil)
- Google Photos partagées avec la famille
- Photos personnelles (réunion d'AG, etc.)
